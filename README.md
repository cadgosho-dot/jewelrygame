## v0.10.113 工房所持工具・設備の購入情報を非表示

- 工房の所持工具・設備詳細から、購入価格と購入日を削除しました。
- 用途、説明、工房評価、現在状態、詳しい説明は引き続き確認できます。
- 故障品と修理中品は、詳細からg-Lab.の受付へ移動できます。
- PWAキャッシュと読み込みバージョンはv0.10.113です。

## v0.10.112 工房所持工具・設備の詳細説明

- 工房の所持工具・設備をタップすると、購入時と同じ構成の専用詳細画面を開きます。
- 用途、説明、工房評価、購入価格、購入日、使用可能・故障中・修理中の状態を確認できます。
- さらに「詳しい説明を見る」から、登録済みの専門的な使用方法・品質管理・安全・保守説明を読めます。
- 故障品と修理中品は、詳細からg-Lab.の受付へ移動できます。
- PWAキャッシュと読み込みバージョンはv0.10.112です。

## v0.10.110 上部バー「メイン画面」ボタン復活

- メイン画面以外の上部バーへ「メイン画面」ボタンを常時表示。
- スマートフォン・小型縦画面・低い横画面でも、戻るボタンや画面タイトルと重ならないよう余白を確保。
- ボタンは既存のメイン画面移動処理を使用し、ゲーム内容やセーブデータは変更していません。
- PWAキャッシュと読み込みバージョンはv0.10.110です。

## v0.10.109 g-Lab.工具・設備カタログ全件表示

- g-Lab.の工具・設備一覧へ、正式登録済みの15種類を常にすべて表示します。
- 工房レベル3・5で解放される品も、未解放の段階から名前・価格・解放条件を確認できます。
- 購入済み、故障中、修理中の品も一覧から消えず、現在状態を表示します。
- 未解放品は詳細を確認できますが、条件を満たすまでは購入できません。
- 名称を「宝石研磨用平面研磨盤」「作業用拡大鏡」「刻印セット」へ正式表記に統一しました。
- PWAキャッシュと読み込みバージョンはv0.10.109です。
