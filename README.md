## v0.10.176 注文品完成後の操作変更

### 注文品が完成した場合
完成画面で選べるのは次の2つだけです。

- お客様へ納品する
- 作り直す

戻るボタンやメイン画面へのボタンは表示されず、どちらかを選ぶまで別画面へ移動できません。

### 作り直す場合
今完成した商品は自動的に通常の保管品となり、注文は未制作の状態へ戻ります。その後、同じ注文内容の制作画面が開きます。

### 通常制作
注文品ではない通常のジュエリー完成画面は従来どおりです。
