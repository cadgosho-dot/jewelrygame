# JEWELRYGAME シンプル基本版 v0.3.8

宝石を採掘し、ジュエリーを作り、店舗で販売するスマートフォン／PC対応のWebゲームです。

## 基本方針

- ゲーム内に固定タイトルやロゴを表示しない
- 目標、ミッション、おすすめ行動を表示しない
- プレイヤーが好きな順番で行動する自由進行型
- 専門的なカット技術や石留め操作は使用しない
- 縦向き・横向きの両方で操作可能
- 端末内へ自動保存
- 主人公のキャラクター選択、プロフィール、立ち絵は使用しない
- ゲーム内のすべての画面で主人公像を表示しない

## 主な機能

- 採掘：河原、山道、鉱山入口
- 宝石6種類
- 地金3種類：シルバー、ゴールド、プラチナ
- ジュエリー制作：リング、ペンダント、ピアス
- デザイン3種類、仕上げ3種類
- 完成品保管、ショーケース、価格設定、自動販売
- 固定客2人、任意の接客、簡単な注文と納品
- 任意の店舗拡張、店員1人
- スマートフォン：注文、予定、通知、収支、設定
- PWA・オフラインキャッシュ

## 公開方法

ZIPを展開し、すべてのファイルをGitHub Pages用リポジトリのルートへアップロードしてください。
