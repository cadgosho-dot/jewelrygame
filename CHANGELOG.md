# CHANGELOG

## v0.3.8 主人公表示の完全削除・背景更新統合

- ゲーム開始時のキャラクターデザイン選択を不採用として確定
- プロフィールから主人公像を完全に削除
- ゲーム内すべての画面で主人公の立ち絵・アバターを表示しない構成へ統一
- 未使用のキャラクターカスタマイズ画像を削除
- 古いセーブデータに主人公関連情報が含まれていても読み込み時に削除
- メイン、店舗、御徒町、採掘、工房、g-Lab.、就寝後の背景画像を最新版へ統合
- Service Workerのキャッシュを更新し、GitHub Pagesで新しい画像が反映されやすいように変更

## v0.3.0 シンプル基本版

- ゲーム全体を自由進行型のシンプル構成へ再設計
- 画面上のゲームタイトル、ロゴ、現在の目標を削除
- 「探索」を「採掘」へ統一
- 強制チュートリアルと解放順を削除
- 主要機能を開始時から利用可能に変更
- 採掘を場所選択＋ボタン操作へ簡略化
- 地金をシルバー、ゴールド、プラチナの3種類へ統一
- ジュエリー制作を5項目の選択方式へ変更
- 品質を標準、良品、上質の3段階へ簡略化
- 完成品、ショーケース、価格設定、自動販売を実装
- 固定客2人、任意接客、簡単な注文と納品を実装
- 任意の店舗拡張と店員1人を実装
- ローカル自動保存へ変更し、ログインを不要化
- 縦向き・横向きのレスポンシブ表示を再構築
