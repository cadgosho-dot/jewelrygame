- 初期所持金を正式仕様どおり30,000円へ修正しました。
- 360×800の縦画面と800×360の横画面を基準に、異なる携帯画面でも縦横比を維持して同じ配置になることを再確認しました。

## v0.10.98 店舗別ショーケース・陳列在庫の分離

### v0.10.98の変更

- 各店舗が独立したショーケースと陳列商品を持ちます。
- 商品は1店舗だけに陳列でき、別店舗へ無料・時間消費なしで移動できます。
- 移動前に設定していた販売価格はそのまま維持されます。
- 接客と販売は、来店した店舗に陳列中の商品だけが対象です。
- 既存セーブのショーケースと陳列商品は店舗1へ引き継ぎます。
- PWAキャッシュと読み込みバージョンをv0.10.98へ更新しました。

## v0.10.95 営業時間整理版

### v0.10.95の変更

- 1日の開始時刻を9:00、作業終了時刻を21:00へ統一。
- 21:00を超える作業を開始できないよう変更。
- 店舗の営業時間を9:00～19:00へ設定。
- 接客、店頭商品提案、注文相談は19:00までに完了できる場合だけ実行可能。
- 注文品の納品は納期当日の19:00まで可能。
- 19:00以降も、商品の陳列、販売価格変更、注文確認、注文品制作、在庫管理は21:00まで利用可能。
- 店舗営業時間終了時に接客中のお客様を退出させ、翌日へ持ち越さないよう変更。
- g-Lab.と御徒町の既存営業時間は変更していない。
- 既存セーブの時刻を9:00～21:00へ安全に補正。
- PWAキャッシュと読み込みバージョンをv0.10.95へ更新。

- 「寝る」の名称と操作
