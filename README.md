## v0.10.138 横画面の表示方式調整

- 横画面は縦の表示領域を基準に画面いっぱいへ表示します。
- 背景は必要に応じて端を切り、下部メニューは端末幅に合わせて均等に伸縮します。
- 固定比率による小さな表示や周囲の余白を防ぎます。
- 縦画面の仕様は維持しています。
- PWAキャッシュと読み込みバージョンはv0.10.138です。

## v0.10.137 端末画面いっぱい表示対応

- 固定比率による縮小表示をやめ、端末で利用できる画面領域いっぱいに表示します。
- 縦横比が異なるスマートフォンでも、大きな白い余白が出ないよう修正しました。
- PWAキャッシュと読み込みバージョンはv0.10.137です。
