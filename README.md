## v0.10.176 注文品完成後の操作変更

### 注文品が完成した場合
完成画面で選べるのは次の2つだけです。

- お客様へ納品する
- 作り直す

戻るボタンやメイン画面へのボタンは表示されず、どちらかを選ぶまで別画面へ移動できません。

### 作り直す場合
今完成した商品は自動的に通常の保管品となり、注文は未制作の状態へ戻ります。その後、同じ注文内容の制作画面が開きます。

### 通常制作
注文品ではない通常のジュエリー完成画面は従来どおりです。

## v0.10.175 次の日の始まり表示時間変更

- 予定やお知らせがない場合、次の日の始まり画面を3秒間表示します。
- その後、約0.3秒のフェードで閉じます。
- 予定やお知らせがある場合は自動では閉じません。

## v0.10.173 デザイン3段階化と表面の選択方法変更

### デザイン
- シンプル
- ノーマル
- ゴージャス

### 表面
- 鏡面
- つや消し
- 装飾あり

鏡面とつや消しはどちらか一方だけ選択できます。装飾ありは単独で選択できるほか、「鏡面＋装飾あり」「つや消し＋装飾あり」の組み合わせにも対応しています。

既存セーブデータの内部IDは維持し、表示名だけを新しい3段階へ更新しています。
